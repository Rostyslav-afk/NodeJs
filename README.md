# NodeJs

Команди:

де я - pwd
подивитись шо є в цій папці - ls
подивитись списком шо в тебе є в папці - ls -l
створити файл - touch
створити папку - mkdir
очистити термінал - clear
зайти кудась - cd   
вийти назад - cd .. 
вийти кудась дальше - cd ../../
видалити файл - rm
видалити папку - rm -r 
перейменувати - mv
